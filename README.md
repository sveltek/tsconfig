<h1 align="center">@sveltek/tsconfig</h1>

<p align="center">Sveltek's config for TypeScript.</p>

<br>

## Installation

```sh
pnpm add -D @sveltek/tsconfig
```

## Usage

### tsconfig.json

```json
{
  "extends": "@sveltek/tsconfig"
}
```
